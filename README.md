# Ταξίδι χωρίς επιστροφή

Ψηφιακή παρουσίαση διδακτικής εφαρμογής για GitHub Pages.

## Πώς το ανεβάζω στο GitHub

1. Δημιουργώ νέο repository στο GitHub.
2. Ανεβάζω όλα τα αρχεία και τον φάκελο `assets`.
3. Πηγαίνω: Settings → Pages.
4. Στο Branch επιλέγω `main` και `/root`.
5. Πατάω Save.
6. Περιμένω 1-2 λεπτά και ανοίγω τον σύνδεσμο που δίνει το GitHub Pages.

## Αρχεία

- `index.html`: η παρουσίαση
- `style.css`: μορφοποίηση
- `script.js`: διαδραστικότητα
- `assets/`: φύλλα εργασίας και κάρτες

## Σημείωση

Οι ιστορικές εικόνες φορτώνονται από Wikimedia Commons μέσω εξωτερικών συνδέσμων. Τα φύλλα εργασίας και οι κάρτες είναι ενσωματωμένα τοπικά στον φάκελο `assets`.
